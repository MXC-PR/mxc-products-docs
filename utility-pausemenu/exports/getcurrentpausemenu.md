# GetCurrentPauseMenu

Return the current pause menu that is being displayed.

<details>

<summary>Returns</summary>

| Type   | Description                |
| ------ | -------------------------- |
| string | "map", "settings", "pause" |

</details>

<details>

<summary>Example</summary>

```lua
local menu = exports["utility_pausemenu"]:GetCurrentPauseMenu()
```

</details>
