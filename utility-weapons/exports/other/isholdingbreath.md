# IsHoldingBreath

Return if the player is holding breath or not

<details>

<summary>Returns</summary>

| Type    | Description                            |
| ------- | -------------------------------------- |
| boolean | If the player is holding breath or not |

</details>

<details>

<summary>Example</summary>

```lua
local breath = exports["utility_weapons"]:IsHoldingBreath()
```

</details>
